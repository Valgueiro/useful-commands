## Iterm2

* `command + option + i` -> broadcast to all panels in one tab


## Linux Terminal

### Screen

* `screen -S name` -> Create
* `crtl + a + d` -> Exit from screen without closing
* `exit` -> Exit quiting it
* `screen -ls` -> List
* `screen -r ID|name` -> Open
* `screen -d ID|name` -> Detach
